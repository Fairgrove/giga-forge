# Deps   
cd to the gigaforge simulations dir   
clone this amazing json reading thingy:   
```bash
curl -O https://raw.githubusercontent.com/nlohmann/json/develop/single_include/nlohmann/json.hpp
```

compile the thing   
```bash
clang++ -std=c++17 -O2 -o giga giga.cpp
```

run it >)
```bash
./giga
```
